# **App Name**: QuizWhiz

## Core Features:

- Quiz Question Generation: AI generates 10 quiz questions based on the topic provided by the user using reasoning capabilities.
- User Topic Input: Allow users to input a topic for quiz generation.
- Quiz Display: Display the quiz questions in a clear, user-friendly format.
- Answer Submission: Allow users to submit answers for each question.
- Score Calculation: Calculate the user's score based on their answers.
- Score Display: Display the user's score at the end of the quiz.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to evoke intellect and focus.
- Background color: Very light lavender (#F0F8FF) for a calming learning environment.
- Accent color: Violet (#8A2BE2) to highlight key elements and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern reading experience.
- Use minimalist icons to represent different quiz categories or feedback elements.
- Employ a clean and organized layout to present questions and options clearly.
- Use subtle animations for feedback and transitions to maintain user engagement.
